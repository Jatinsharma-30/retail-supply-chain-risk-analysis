# Notebooks Directory

## Purpose

This folder contains all Jupyter notebooks used for data cleaning, feature engineering, exploratory data analysis, and business insight generation.

## Notebook Structure

### 01_data_cleaning.ipynb

Responsible for:

- Data loading
- Missing value treatment
- Duplicate removal
- Data type correction
- Date conversion
- Feature engineering

### 02_eda.ipynb

Responsible for:

- Descriptive statistics
- Trend analysis
- Inventory analysis
- Forecast analysis
- Seasonal analysis
- Visual exploration

## Expected Outputs

- Clean dataset
- Business insights
- Visualizations
- Analytical observations
