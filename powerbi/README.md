# Power BI Directory

## Purpose

This folder contains Power BI dashboards developed for business stakeholders.

## Dashboard Pages

### Executive Overview

Key KPIs and overall business performance.

### Inventory Dashboard

Inventory monitoring and stock analysis.

### Sales Dashboard

Regional and category-level sales analysis.

### Demand Forecast Dashboard

Forecast accuracy and demand evaluation.

### External Factors Dashboard

Impact of seasonality, weather conditions, competitor pricing, and promotions.

## Expected Outcomes

- Better inventory planning
- Improved forecasting visibility
- Enhanced business decision-making
