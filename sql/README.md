# SQL Directory

## Purpose

This folder contains all SQL scripts used for business analysis and KPI generation.

## Analysis Areas

### Inventory Analysis

- Low inventory products
- Inventory value calculation
- Category-wise inventory analysis

### Sales Analysis

- Region-wise sales
- Product performance
- Category performance

### Demand Forecast Analysis

- Forecast error calculation
- Demand accuracy measurement

### Advanced SQL Concepts

The project demonstrates:

- CTEs
- Window Functions
- RANK()
- DENSE_RANK()
- LAG()
- ROLLUP()
- CASE Statements

## Files

inventory_analysis.sql

Contains all analytical SQL queries used in the project.
